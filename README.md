# Geo

A WebGPU (a possibly others) 3D engine.

### Running

`deno task start`

### Testing

`deno test`

### Controls

- WASD translation
- Arrows rotation
- Mouse drag tumble
- Shift + R toggle video recording

To start: `deno task start`
